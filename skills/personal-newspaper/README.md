# Personal Newspaper Codex Skill

This distributable skill operates one deployed Personal Newspaper site. Copy this folder into the Codex skills directory or keep it in this repository for project-local use.

Requirements: a deployed site URL, the site `AUTOMATION_TOKEN`, and an owner-approved Codex Automation. Use `pnpm edition:publish -- --file edition.json --url "$PAPER_URL" --token "$AUTOMATION_TOKEN"` to upload an already verified edition bundle.

The full setup, update, publish-now, and status contract is in [SKILL.md](SKILL.md).
